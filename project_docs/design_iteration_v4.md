# Design Iteration: Stepper Controls

To eliminate manual typing for quantities, I've replaced the text inputs with **Stepper Controls**.

## Mobile Item Card v2
*   **Min Qty** and **Max Qty** now feature large **[- 1 +]** buttons.
*   This makes it easy to adjust values with a single tap, which is much faster on mobile than bringing up the keyboard.
